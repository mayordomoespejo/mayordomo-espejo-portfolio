# Mayordomo Espejo Portfolio

Portfolio basado en Next.js. Este es tu propio proyecto; no está vinculado al repositorio original.

## Personalización rápida

1. **Nombre y metadata**: `app/layout.tsx` — título y descripción.
2. **Hero**: `components/hero.tsx` — tu nombre y foto (sustituye `public/images/profile.jpg`).
3. **Contacto**: `components/contact-cta.tsx` y `components/footer.tsx` — sustituye `tu-email@ejemplo.com` y `https://www.linkedin.com/in/tu-perfil/` por tu email y LinkedIn.
4. **About**: `components/about-content.tsx` — tu historia, skills y educación.
5. **Proyectos**: `lib/projects.ts` — edita o reemplaza los proyectos con los tuyos.

## Desarrollo

```bash
pnpm install
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```
